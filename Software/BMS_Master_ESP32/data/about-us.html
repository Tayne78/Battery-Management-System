BMS

Battery Management System


Dieses Battery Management System entstand im Zuge einer Diplomarbeit an der HTL St. Pölten. Die Aufgabe der Diplomanden Thomas Kaiser und Markus Tucek war es unter Betreuung von Dr. DI Reinhard Steindl ein System zu entwickeln welches das gefahrenlose Betreiben eines Elektrofahrzeuges erlaubt.

Das Projekt entstand unter Zusammenarbeit mit der Elektrotechnik Abteilung welche einen VW-Käfer zu einen Elektrofahrzeug umbauten. Wofür auch das BMS verwendet werden sollte.

Das gesamte Projekt ist Open Source und kann auf GitHub eingesehen werden.
